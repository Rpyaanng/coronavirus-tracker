# CoronaVirus Tracker

CoronaVirus Tracker is a simple application that I made using Spring Boot.

## Installation

[JDK 17+](https://www.oracle.com/java/technologies/downloads/#java18) is needed to build.

```bash
./mvnw spring-boot:run
```
